= Basic Styling
Format **basic text styles** like this:
```
**bold text**
//italic text//
`monospaced text`
##monospaced text##
~~deleted text~~
__underlined text__
!!highlighted text!!
```

Those produce **bold text**, //italic text//, `monospaced text`, ##monospaced text##,
~~deleted text~~, __underlined text__, and !!highlighted text!!
respectively.
